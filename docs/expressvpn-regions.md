# ExpressVPN Region Slugs

Captured from the installed Linux CLI with:

```bash
expressvpnctl get regions
```

The command returned 212 selectable region slugs, including `smart`.
Use these values as the ExpressVPN sidecar `EXPRESSVPN_LOCATION` or as egress
profile metadata `region` values. Availability can change over time.

## High-Use Country Examples

- United States: `usa-san-francisco`, `usa-new-york`, `usa-chicago`, `usa-los-angeles-1`
- Japan: `japan-yokohama`, `japan-tokyo`, `japan-shibuya`, `japan-osaka`
- South Korea: `south-korea-2`
- Singapore: `singapore-cbd`, `singapore-jurong`, `singapore-marina-bay`
- United Kingdom: `uk-london`, `uk-docklands`, `uk-manchester`, `uk-wembley`
- Germany: `germany-frankfurt-1`, `germany-frankfurt-3`, `germany-berlin`, `germany-nuremberg`
